# WuKongIM Docs Site

This directory contains the new WuKongIM documentation site scaffold built with
Fumadocs, React Router, and Vite.

## Commands

Install dependencies:

```bash
npm install
```

Start local development:

```bash
npm run dev
```

Build static output:

```bash
npm run build
```

Preview the built site:

```bash
npm run preview
```

## Output

- Development site runs from the project root path `/`
- Static files are generated into `dist/client/`
- Current placeholder pages are `/` and `/getting-started`
