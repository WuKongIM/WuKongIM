---
title: 发送第一条消息
description: 用一个最小可感知的流程证明 WuKongIM 已经成功跑通。
---

最简单的验证方式不是先写代码，而是先让两个用户真正收发一条消息。

## 方式一：直接用聊天 Demo

打开：

```text
http://127.0.0.1:5172/chatdemo
```

然后做下面几步：

1. 用用户 A 登录一个浏览器窗口
2. 用用户 B 登录另一个窗口
3. 互发一条文本消息

如果消息能即时到达，说明下面这些链路已经打通：

- 客户端连接
- 会话建立
- 消息发送
- 消息投递

## 方式二：用 Web SDK 最小初始化

如果你想立刻进入代码侧验证，可以先看 README 里的最小接入思路：

```ts
import { WKIM, WKIMChannelType } from 'easyjssdk';

const im = WKIM.init('ws://your-wukongim-server.com:5200', {
  uid: 'your_user_id',
  token: 'your_auth_token',
});
```

在连接成功后，再发送一条消息即可。

## 看到什么才算成功

- 连接建立成功，没有持续重连
- 发送后另一端立即收到
- 管理后台能看到连接或消息变化

## 如果失败，先回查这些点

- `5200` 是否可访问
- `WK_EXTERNAL_IP` 是否配置正确
- 客户端用的是不是正确的 `uid` 和 `token`
- 你是否误把 `5001` 当成了长连接地址
