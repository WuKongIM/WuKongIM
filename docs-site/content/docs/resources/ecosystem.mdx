- 官网 / 文档：`https://githubim.com`
- 主仓库：`https://github.com/WuKongIM/WuKongIM`
- 架构文档：`https://deepwiki.com/WuKongIM/WuKongIM`

如果你只记三个链接，先记这三个。

## 怎么使用这页

### 适合拿来找链接，不适合从头顺读

这页的价值在于“快速跳转”，不是“系统学习”。  
如果你发现自己开始一条条点链接，通常说明应该先回主路径，而不是继续留在资源区。

### 想做深度研究，再回来看这些资料

- 协议说明
- 架构文档
- 压测报告
- 发布记录
