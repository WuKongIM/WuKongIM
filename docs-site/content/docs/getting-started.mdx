This page is a placeholder for the first version of the documentation site.

## Local Development

Install dependencies and start the dev server:

```bash
npm install
npm run dev
```

The site should be available locally with the docs homepage mounted at `/`.

## Production Build

Create the static output:

```bash
npm run build
```

The generated files are written under `docs-site/dist`.
