Web SDK 是当前新文档里最适合作为“完整 SDK”阅读样板的平台。

如果你要做的是：

- Web 聊天应用
- 管理后台内嵌 IM
- 客服工作台
- H5 或桌面网页消息能力

那么这组文档可以直接作为主路径来读。

## Web SDK 和快速接入版有什么区别

快速接入更像“先证明链路通了”，而 Web 完整 SDK 更偏“我要把完整客户端能力接起来”。

它通常会多出这些事情：

- 自己维护连接生命周期
- 对接消息同步数据源
- 对接最近会话数据源
- 管理频道资料和成员缓存
- 处理自定义消息、状态监听和业务扩展
