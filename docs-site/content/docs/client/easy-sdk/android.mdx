Android 的最短接入路径同样只需要四步：安装、初始化、监听、连接。

## 第一步：安装 SDK

Gradle 示例：

```kotlin
dependencies {
    implementation("com.githubim:easysdk-android:1.0.0")
}
```

## 第二步：初始化

```kotlin
val config = WuKongConfig.Builder()
    .serverUrl("ws://your-wukongim-server.com:5200")
    .uid("your_user_id")
    .token("your_auth_token")
    .build()

val easySDK = WuKongEasySDK.getInstance()
easySDK.init(this, config)
```

## 第三步：监听事件

```kotlin
easySDK.addEventListener(WuKongEvent.CONNECT) { result ->
    Log.d("WuKong", "Connected: $result")
}

easySDK.addEventListener(WuKongEvent.MESSAGE) { message ->
    Log.d("WuKong", "Received: $message")
}
```

## 第四步：建立连接

```kotlin
easySDK.connect()
```

## 最常见的错误

- Activity 销毁后没有移除监听器
- WebSocket 地址写成了 API 地址
- 登录流程里没有先同步好 `uid` / `token`
