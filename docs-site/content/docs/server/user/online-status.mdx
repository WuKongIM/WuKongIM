---
title: 获取在线状态
description: 批量查询用户是否在线，适合做状态展示、在线路由或业务判断。
---

核心接口：

```text
POST /user/onlinestatus
```

## 请求体格式

直接传 UID 数组：

```json
["user1", "user2", "user3"]
```

## 返回内容

每个用户会返回：

- `uid`
- `online`
- `device_flag`

其中：

- `online=0` 表示离线
- `online=1` 表示在线

## 最小示例

```bash
curl -X POST "http://localhost:5001/user/onlinestatus" \
  -H "Content-Type: application/json" \
  -d '["user1", "user2", "user3"]'
```

## 适合场景

- 联系人在线状态展示
- 消息投递前做简单在线判断
- 管理后台查看用户在线情况
