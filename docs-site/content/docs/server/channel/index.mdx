频道是 WuKongIM 里最核心的业务对象之一。你可以把它理解成“消息投递和成员关系发生的地方”。

服务端最常见的频道操作只有三类：

- 创建频道
- 更新频道信息
- 管理频道成员

## 什么时候会用到频道接口

- 业务里要创建群组或业务频道
- 需要修改频道禁言、发送权限、陌生人消息能力
- 需要批量添加或移除成员
- 需要做临时订阅者或成员重置

## 频道接口怎么读

### 先分清频道类型

- `1`：个人频道
- `2`：群组频道

### 再决定你是在改“频道属性”还是改“成员关系”

- 频道属性：禁言、解散、是否允许陌生人发送等
- 成员关系：添加成员、移除成员、临时订阅者、重置成员列表
