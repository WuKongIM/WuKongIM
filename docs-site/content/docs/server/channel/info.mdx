---
title: 更新频道信息
description: 当频道已经存在时，用这个接口调整禁言、发送权限和其他基础属性。
---

更新频道信息对应的核心接口是：

```text
POST /channel/info
```

## 适合修改哪些内容

- `ban`：是否全员禁言
- `disband`：是否解散频道
- `send_ban`：是否禁止发送消息
- `allow_stranger`：个人频道是否允许陌生人发消息

## 最小示例

```bash
curl -X POST "http://localhost:5001/channel/info" \
  -H "Content-Type: application/json" \
  -d '{
    "channel_id": "group123",
    "channel_type": 2,
    "ban": 1
  }'
```

## 使用建议

- 先保证频道已存在
- 明确这是在改“频道属性”，不是改成员关系
- 频道成员变化请走成员管理接口，不要混到这里

## 什么时候需要它

- 群聊切换全员禁言
- 业务侧禁用某个频道发言
- 调整个人频道的陌生人消息策略
