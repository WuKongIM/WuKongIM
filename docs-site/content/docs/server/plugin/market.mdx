## 插件包命名规则

插件格式：

`[插件名字]-[系统]-[CPU架构].wkp`

例如：

- `wk.plugin.ai-example-linux-amd64.wkp`
- `wk.plugin.ai-example-darwin-amd64.wkp`

常见 CPU 架构为 `amd64`。

## 下载时怎么选

按你部署 WuKongIM 的实际运行环境选择对应插件包：

- `系统` 要和服务器操作系统一致
- `CPU 架构` 要和服务器架构一致

如果选择错误，插件通常无法正常执行。
