WuKongIM is a distributed instant messaging server focused on high performance,
high availability, and large-scale messaging scenarios.

This site is the new documentation skeleton for the project. The first version
keeps the content intentionally small so the docs site can run locally and build
cleanly before the existing material is migrated over.
